# EmillioID
<img width="2433" height="1097" alt="image" src="https://github.com/user-attachments/assets/c61daf12-953e-447f-9f6d-74220c4fbcff" />

EmillioID is the identity provider for public systems across Emillio Releases and its subsidiaries. This system is recommended to not be used in production as it isn't fully tested for sensitive data use.

## TECH STACK
- Next.js
- Turso
- Auth.js
- Cloudflare Workers
